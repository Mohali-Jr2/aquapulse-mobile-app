# AquaPulse Smart Water User App

A modern Flutter mobile application for a Smart Water Metering System.

## Main Features

- Splash and welcome screens
- User login and registration UI
- Smart dashboard with real-time usage mock data
- Live usage monitoring
- Usage analytics and consumption trends
- Bills and mobile money payment UI
- Alerts and notifications center
- Valve control screen
- Reports screen
- Profile and linked meter details
- Support screen

## How to Run

```bash
flutter pub get
flutter run
```

## Project Structure

```text
lib/
  core/
    constants/
    theme/
  data/
  models/
  screens/
  services/
  widgets/
```
